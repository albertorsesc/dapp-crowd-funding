### Installation

Clone Repo

```shell
git clone https://github.com/albertorsesc/dapp-crowd-funding.git
```

Install NPM dependencies

```shell
$ cd web3
$ npm i

$ cd ../client
$ npm i
```

Deploy Smart Contract to your Thirdweb account

```shell
$ cd web3
$ npx thirdweb@latest deploy
```

Start dApp

```shell
$ cd client
$ npm run dev
```


### Requirements

- [Thirdweb](https://thirdweb.com/) account.
